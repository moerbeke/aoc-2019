# Advent of Code 2019 solver

## Run solver for day 1

The input is expected to be in a text file called `1.in`.

Run tests of _day 1_:

```bash
./test_aoc1.py
```

Solve _day 1_, part 1:

```bash
./aoc1.py -p1
```

Solve _day 1_, part 2:

```bash
./aoc1.py -p2
```

Solve _day 1_, both parts:

```bash
./aoc1.py
```
